# User Onboarding Flow

A full-stack web application that takes a new user through a complete signup and onboarding experience.

## Features

- User signup and login with hashed passwords (bcrypt)
- JWT-based authentication
- Simulated payment step
- 3-question onboarding form
- Protected routes — users can't skip steps
- Personal dashboard showing user data
- Data privacy — users only see their own data

## Tech Stack

- **Backend:** Node.js + Express
- **Frontend:** React (Vite)
- **Database:** SQLite (via better-sqlite3)
- **Auth:** bcrypt + JWT
